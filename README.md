## Setup

1. Clone the repo:

```sh
    git clone https://github.com/Ermegilius/FullStack_Storage_and_Booking_App.git
```

2. Change directory to the project folder:

```sh
    cd FullStack_Storage_and_Booking_App
```

3. Install dependencies:

```sh
    npm install
```

4. Run the app:

```sh
    npm start
```
